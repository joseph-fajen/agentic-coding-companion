# Agentic Coding Companion

**[View the Guide →](https://joseph-fajen.github.io/agentic-coding-companion/)**

A scannable reference guide for the [Dynamous](https://dynamous.ai/) agentic coding workflow.

## What This Is

I created this as a personal quick-reference for the methods taught in the Dynamous Agentic Coding course. Rather than scrubbing through video lectures, I wanted a fast way to verify I'm applying the methodology correctly at each workflow stage.

## How It Was Made

This project demonstrates using AI to transform existing assets into structured documentation:

1. **Source material** — Course lecture transcripts (~40,000 lines)
2. **Technical writing** — Distilled transcripts into scannable principles and examples
3. **Product requirements** — Created a [PRD](PRD.md) defining scope, structure, and user needs
4. **AI-assisted build** — Used the PRD to guide AI in generating the Astro/Starlight docs site

The result is six workflow stages — Setup → Planning → Implementation → Validation → Iteration — each with quick-reference summaries and copyable command patterns.

## Sharing

I shared this with the Dynamous community and received positive feedback. If you're taking the course or practicing agentic coding, you're welcome to use it as a reference alongside the official course materials.
